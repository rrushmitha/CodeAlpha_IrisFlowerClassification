
# Iris Flower Classification

## Objective
To classify Iris flowers into Setosa, Versicolor and Virginica using Machine Learning.

## Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## Model
Random Forest Classifier

## Results
- Accuracy: 100%
- Confusion Matrix Generated
- Pair Plot Visualization
- Correlation Heatmap

## Conclusion
The model successfully classified Iris flower species with high accuracy.
